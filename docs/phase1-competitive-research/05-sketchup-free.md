# Competitor Analysis: SketchUp Free (sketchup.com)

## Target Audience
Hobbyists, makers, students, and professionals needing quick 3D visualization. Least consumer-focused of the group.

## Pros
- Completely free web-based 3D modeling
- Massive 3D Warehouse with user-contributed models
- 10 GB Trimble Connect cloud storage
- Push/Pull methodology makes 3D from 2D intuitive once learned
- STL export for 3D printing (unique among competitors)
- Extremely powerful and flexible for custom geometry
- Strong learning resources (blog, training, forums)

## Cons / Weaknesses
- ZERO floorplan digitization -- no AI, no assisted tracing, no floorplan tools at all
- No CAD-lite workflow -- it's a full 3D modeler for architectural modeling
- Steepest learning curve of all competitors
- No automatic wall snapping, no door/window tools, no room dimensioning
- Free version lacks DWG/DXF import (paid feature only)
- No furniture drag-and-drop library -- manual 3D Warehouse search for each item
- No 2D export with dimensions for floorplan-style outputs
- Web-only performance is poor for complex scenes
- Completely overkill for the HomeForge use case

## Pricing Model
Free web version. Paid tiers (SketchUp Go, Pro, Studio) for advanced features.

## Floorplan Digitization
None. You import a JPG/PNG as a flat reference image and manually draw everything over it from scratch. No assistance whatsoever.

## Key Takeaway for HomeForge
SketchUp Free proves there is demand for free 3D design in the browser, but it is entirely unsuited for our use case. The only lesson: 3D Warehouse shows the value of a large model library, but for our MVP, a curated set of ~40 common items is vastly preferable to an overwhelming searchable catalog.
