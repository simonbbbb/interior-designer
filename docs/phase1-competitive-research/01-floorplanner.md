# Competitor Analysis: Floorplanner (floorplanner.com)

## Target Audience
Prosumer -- leans consumer but heavily used by real estate agents and interior designers for quick presentations.

## Pros
- 260,000+ 3D models in their library (massive catalog)
- High-quality 3D renders up to 8K resolution
- "3D Dollhouse" views for immersive presentations
- Free tier is genuinely usable for basic projects
- Educational accounts are completely free
- Credit-based system allows pay-per-use instead of forced subscriptions
- Fast and performant in-browser 2D/3D editor
- Clean, approachable interface

## Cons / Weaknesses
- Floorplan digitization is purely MANUAL -- users must upload an image and trace walls by hand. No AI auto-detection.
- Wall editing after tracing requires manual snapping and alignment -- walls don't auto-join or auto-detect intersections well
- The 2D editor feels dated (mid-2010s UI patterns)
- Exporting high-res renders requires credits (paywall)
- Furniture items are generic catalog items -- not real-world dimensioned
- No collaboration or real-time co-editing
- Mobile experience is essentially read-only

## Pricing Model
- Free for casual use ("Basic account forever")
- One-time credits for project upgrades
- Optional subscriptions for frequent users
- Free for all schools worldwide

## Floorplan Digitization
Manually trace over uploaded images. No AI, no auto-detection. Users draw walls by hand on top of the imported image.

## Key Takeaway for HomeForge
Floorplanner has the best-performing canvas editor but lacks any AI digitization. Wall tracing is tedious. We should borrow their performance characteristics but add the AI layer they're missing.
