# Keyboard Shortcuts

| Shortcut | Action | Notes |
|---|---|---|
| Meta+Z | Undo | Cmd on Mac, Ctrl on Windows/Linux |
| Meta+Shift+Z | Redo | Alternative |
| Meta+Y | Redo | Windows/Linux preferred |
| Delete / Backspace | Delete selected | Only when object selected |
| R | Rotate 90 degrees | When furniture selected |
| Arrow keys | Nudge 1px | Selected object |
| Shift+Arrow | Nudge 10px | Selected object |
| W | Draw wall tool | When no input focused |
| S | Select tool | When no input focused |
| Escape | Deselect / cancel | |
| Meta+S | Save project | |
| Meta+E | Export project | |
| Space (hold) | Temporary pan | Hold to pan, release to resume tool |

Single-key shortcuts only work when:
- No text input is focused
- No modal is open
- Canvas is the active element
